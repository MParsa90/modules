کتابخانه‌های پایتون برای دانش‌آموزان

سه کتابخانه کاربردی برای محاسبات ریاضی و رابط گرافیکی - مدرسه علامه‌حلی یک

به کتابخانه‌های پایتون خوش آمدید

این مجموعه شامل سه کتابخانه قدرتمند پایتون است که به صورت اختصاصی برای دانش‌آموزان مدرسه علامه‌حلی یک طراحی شده‌اند. هر کتابخانه با هدف ساده‌سازی مفاهیم پیچیده ریاضی و برنامه‌نویسی ایجاد شده و به شما کمک می‌کند تا با لذت و سرعت بیشتری کدنویسی کنید.

کتابخانه‌های موجود

جدید helliemod - کتابخانه جامع ریاضیات

کتابخانه‌ای کامل برای انجام محاسبات ریاضی از پایه تا پیشرفته شامل توابع فاکتوریل، اعداد اول، فیبوناچی و بسیاری عملیات ریاضی دیگر

pip install helliemod

محبوب fruitmod - ریاضیات با طعم میوه!

کتابخانه خلاقانه‌ای که نام توابع ریاضی را با نام میوه‌ها جایگزین کرده است. هم آموزشی، هم سرگرم‌کننده و هم قدرتمند

pip install fruitmod

کاربردی humblytkinter - رابط کاربری آسان

کتابخانه‌ای برای ساده‌سازی ساخت رابط‌های گرافیکی با tkinter. ایجاد پنجره، دکمه و فرم‌ها تنها با چند خط کد ساده

pip install humblytkinter
نکته مهم: تمام این کتابخانه‌ها به صورت رایگان در اختیار دانش‌آموزان مدرسه علامه‌حلی قرار دارد و برای استفاده آموزشی طراحی شده‌اند. برای دسترسی به مستندات کامل هر کتابخانه، روی نام آن در منوی بالا کلیک کنید.

کتابخانه helliemod

کتابخانه helliemod یک ابزار جامع برای انجام محاسبات ریاضی است که به صورت اختصاصی برای دانش‌آموزان طراحی شده است. این کتابخانه شامل توابع متعددی است که انجام محاسبات ریاضی را ساده‌تر و سریع‌تر می‌کند.

نصب کتابخانه

برای نصب این کتابخانه از دستور pip استفاده کنید:

pip install helliemod

توابع اصلی

fact(n)
محاسبه فاکتوریل یک عدد
مثال ۱: فاکتوریل عدد ۵
import helliemod result = helliemod.fact(5) print(result) # خروجی: 120
مثال ۲: فاکتوریل عدد ۰
import helliemod result = helliemod.fact(0) print(result) # خروجی: 1 (طبق تعریف)
مثال ۳: فاکتوریل عدد ۱۰
import helliemod result = helliemod.fact(10) print(f"10! = {result}") # خروجی: 3628800
compare(a, b)
مقایسه دو عدد و بازگرداندن عدد بزرگ‌تر
مثال ۱: مقایسه ۱۵ و ۲۳
import helliemod max_num = helliemod.compare(15, 23) print(max_num) # خروجی: 23
مثال ۲: مقایسه اعداد منفی
import helliemod max_num = helliemod.compare(-5, -10) print(max_num) # خروجی: -5
مثال ۳: مقایسه اعداد مساوی
import helliemod max_num = helliemod.compare(7, 7) print(max_num) # خروجی: 7
check_prime(n)
بررسی اول بودن یک عدد
مثال ۱: بررسی عدد ۱۷
import helliemod is_prime = helliemod.check_prime(17) print(is_prime) # خروجی: True
مثال ۲: بررسی عدد ۱
import helliemod is_prime = helliemod.check_prime(1) print(is_prime) # خروجی: False
مثال ۳: بررسی عدد ۱۰۰
import helliemod is_prime = helliemod.check_prime(100) print(is_prime) # خروجی: False
fib_seq(n)
تولید دنباله فیبوناچی تا عدد nام
مثال ۱: دنباله ۸ عددی
import helliemod fib_sequence = helliemod.fib_seq(8) print(fib_sequence) # خروجی: [0, 1, 1, 2, 3, 5, 8, 13]
مثال ۲: دنباله ۵ عددی
import helliemod fib_sequence = helliemod.fib_seq(5) print(fib_sequence) # خروجی: [0, 1, 1, 2, 3]
مثال ۳: دنباله ۱ عددی
import helliemod fib_sequence = helliemod.fib_seq(1) print(fib_sequence) # خروجی: [0]
gcd(a, b)
محاسبه بزرگ‌ترین مقسوم‌علیه مشترک
مثال ۱: ب.م.م ۴۸ و ۱۸
import helliemod gcd_val = helliemod.gcd(48, 18) print(gcd_val) # خروجی: 6
مثال ۲: ب.م.م ۱۷ و ۵
import helliemod gcd_val = helliemod.gcd(17, 5) print(gcd_val) # خروجی: 1
مثال ۳: ب.م.م ۱۰۰ و ۷۵
import helliemod gcd_val = helliemod.gcd(100, 75) print(gcd_val) # خروجی: 25
lcm(a, b)
محاسبه کوچک‌ترین مضرب مشترک
مثال ۱: ک.م.م ۱۲ و ۱۵
import helliemod lcm_val = helliemod.lcm(12, 15) print(lcm_val) # خروجی: 60
مثال ۲: ک.م.م ۷ و ۵
import helliemod lcm_val = helliemod.lcm(7, 5) print(lcm_val) # خروجی: 35
مثال ۳: ک.م.م ۱۰ و ۲۵
import helliemod lcm_val = helliemod.lcm(10, 25) print(lcm_val) # خروجی: 50
ویژگی‌های پیشرفته: این کتابخانه همچنین شامل تابع help() است که راهنمای کاملی از تمام توابع را نمایش می‌دهد. برای استفاده کافیست بنویسید: helliemod.help()

کتابخانه fruitmod

کتابخانه fruitmod یک رویکرد خلاقانه برای آموزش مفاهیم ریاضی ارائه می‌دهد. در این کتابخانه، هر تابع ریاضی با نام یک میوه مرتبط شده است که یادگیری و به خاطر سپردن توابع را ساده‌تر و لذت‌بخش‌تر می‌کند.

نصب کتابخانه

برای نصب این کتابخانه از دستور pip استفاده کنید:

pip install fruitmod

توابع و معادل‌های میوه‌ای

sib(n) / factorial(n)
سیب (sib): محاسبه فاکتوریل عدد
مثال ۱: فاکتوریل ۵ با نام میوه‌ای
import fruitmod result = fruitmod.sib(5) print(result) # خروجی: 120
مثال ۲: فاکتوریل ۳ با نام معمولی
import fruitmod result = fruitmod.factorial(3) print(result) # خروجی: 6
مثال ۳: ترکیب نام‌ها
import fruitmod if fruitmod.khiar(7): # بررسی اول بودن result = fruitmod.sib(4) # فاکتوریل print(f"4! = {result}")
golabi(a, b) / sum(a, b)
گلابی (golabi): محاسبه مجموع اعداد بین a و b
مثال ۱: مجموع ۱ تا ۱۰
import fruitmod total = fruitmod.golabi(1, 10) print(total) # خروجی: 55
مثال ۲: مجموع ۵ تا ۱۵
import fruitmod total = fruitmod.sum(5, 15) print(total) # خروجی: 110
مثال ۳: مجموع اعداد منفی
import fruitmod total = fruitmod.golabi(-5, 5) print(total) # خروجی: 0
tameshk(a, b) / compare(a, b)
تمشک (tameshk): مقایسه دو عدد و بازگرداندن عدد بزرگتر
مثال ۱: مقایسه ۸ و ۳
import fruitmod larger = fruitmod.tameshk(8, 3) print(larger) # خروجی: 8
مثال ۲: مقایسه ۱۵ و ۲۳
import fruitmod larger = fruitmod.compare(15, 23) print(larger) # خروجی: 23
مثال ۳: مقایسه اعداد منفی
import fruitmod larger = fruitmod.tameshk(-5, -10) print(larger) # خروجی: -5
ananas(n) / fibonacci(n)
آناناس (ananas): محاسبه عدد nام در دنباله فیبوناچی
مثال ۱: عدد هفتم فیبوناچی
import fruitmod fib = fruitmod.ananas(7) print(fib) # خروجی: 13
مثال ۲: عدد دهم فیبوناچی
import fruitmod fib = fruitmod.fibonacci(10) print(fib) # خروجی: 55
مثال ۳: عدد اول فیبوناچی
import fruitmod fib = fruitmod.ananas(1) print(fib) # خروجی: 1
khiar(n) / check_prime(n)
خیار (khiar): بررسی اول بودن عدد
مثال ۱: بررسی عدد ۱۱
import fruitmod is_prime = fruitmod.khiar(11) print(is_prime) # خروجی: True
مثال ۲: بررسی عدد ۱۰۰
import fruitmod is_prime = fruitmod.check_prime(100) print(is_prime) # خروجی: False
مثال ۳: بررسی عدد ۲
import fruitmod is_prime = fruitmod.khiar(2) print(is_prime) # خروجی: True
portaghal(a, b) / gcd(a, b)
پرتقال (portaghal): محاسبه بزرگترین مقسوم علیه مشترک دو عدد
مثال ۱: ب.م.م ۴۸ و ۱۸
import fruitmod gcd_val = fruitmod.portaghal(48, 18) print(gcd_val) # خروجی: 6
مثال ۲: ب.م.م ۳۶ و ۲۴
import fruitmod gcd_val = fruitmod.gcd(36, 24) print(gcd_val) # خروجی: 12
مثال ۳: ب.م.م ۱۷ و ۵
import fruitmod gcd_val = fruitmod.portaghal(17, 5) print(gcd_val) # خروجی: 1
نکته آموزشی: این کتابخانه به گونه‌ای طراحی شده که دانش‌آموزان همزمان با نام‌های میوه‌ای و نام‌های استاندارد ریاضی آشنا شوند. توصیه می‌شود در مراحل اولیه از نام‌های میوه‌ای استفاده کنید و به تدریج به سمت نام‌های استاندارد بروید.

کتابخانه humblytkinter

کتابخانه humblytkinter ایجاد رابط‌های کاربری گرافیکی را به ساده‌ترین شکل ممکن می‌کند. این کتابخانه بر پایه tkinter ساخته شده اما تمام پیچیدگی‌های آن را حذف کرده است.

نصب کتابخانه

برای نصب این کتابخانه از دستور pip استفاده کنید:

pip install humblytkinter

توابع اصلی

make_window(title)
ایجاد یک پنجره جدید با عنوان مشخص
مثال ۱: ایجاد پنجره ساده
import humblytkinter as ht # ساخت پنجره window = ht.make_window("برنامه من") ht.last_code()
مثال ۲: پنجره با عنوان فارسی
import humblytkinter as ht # ساخت پنجره با عنوان فارسی window = ht.make_window("ماشین حساب") ht.last_code()
مثال ۳: پنجره با عنوان طولانی
import humblytkinter as ht # ساخت پنجره با عنوان طولانی window = ht.make_window("برنامه مدیریت دانش آموزان") ht.last_code()
make_label(text, number)

مثال ۲: چندین برچسب
import humblytkinter as ht ht.make_window("برنامه آموزشی") ht.make_label("نام دانش آموز:", number=1) ht.make_label("نام خانوادگی:", number=2) ht.make_label("پایه تحصیلی:", number=3) ht.last_code()

مثال ۲: دکمه محاسبه

make_entry(text, number)

get_output(number)
دریافت مقدار وارد شده در ویجت
مثال ۱: دریافت مقدار فیلد
import humblytkinter as ht def process(): username = ht.get_output(1) print(f"نام کاربری: {username}") ht.make_window("ورودی نمونه") ht.make_entry("", number=1) ht.make_button("ثبت", command=process, number=1) ht.last_code()

import humblytkinter as ht ht.make_window("برنامه کامل") # تعریف چندین ویجت ht.make_label("عنوان برنامه", number=1) ht.make_entry("ورودی متن", number=1) ht.make_button("اجرا", command=some_function, number=1) ht.last_code() # نمایش رابط کاربری
نکات مهم:
  • شماره‌گذاری ویجت‌ها باید منحصر به فرد باشد
  • توابع رویداد (command) را بدون پرانتز وارد کنید
  • تابع last_code() باید همیشه در انتهای برنامه فراخوانی شود
