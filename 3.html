راهنمای استفاده از ماژول fruitmod

نصب ماژول

برای نصب ماژول، کافیست در ترمینال بنویسید:

pip install fruitmod

وارد کردن در برنامه

برای استفاده از توابع ماژول:

import fruitmod

توابع اصلی

  • sib(n) یا factorial(n): محاسبه فاکتوریل
  • golabi(a, b) یا sum(a, b): مجموع اعداد بین دو عدد
  • tameshk(a, b) یا compare(a, b): عدد بزرگ‌تر
  • porteghal(n) یا perfect_square(n): نزدیک‌ترین مربع کامل
  • talebi(n) یا sum_digits(n): شمارش ارقام
  • havij(n) یا reverse(n): مقلوب عدد
  • kivi(n) یا perfect(n): بررسی عدد کامل
  • khiar(n) یا check_prime(n): بررسی اول بودن
  • holoo(a, b) یا gcd(a, b): بزرگ‌ترین مقسوم‌علیه مشترک
  • ananas(n) یا fibonacci(n): عدد nام دنباله فیبوناچی
  • aloo(n, a) یا count_digits(n, a): شمارش رقم مشخص
  • moz(n) یا count_divisor(n): تعداد مقسوم‌علیه‌ها
  • anar(n) یا absolute(n): قدر مطلق
  • angoor(n) یا print_reverse(n): چاپ ارقام از راست به چپ
  • albaloo(a, b) یا print_atob(a, b): چاپ بازه اعداد

راهنمای گرافیکی

برای دیدن لیست توابع به صورت گرافیکی از:

fruitmod.help()

استفاده کنید.
